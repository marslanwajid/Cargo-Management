About the System

The Cargo Management System is a comprehensive solution for managing cargo-related operations. Developed using a full-stack approach with HTML, CSS, JavaScript, PHP, and SQL, this software provides a user-friendly interface and robust functionality for handling various aspects of cargo management.

Key Features

Super Admin Dashboard

  • Exclusive Access: The Super Admin has complete control over the system, with the ability to manage admin accounts, view and delete invoices, and monitor overall system activities.
  • Admin Management: Add, update, or remove admin users, ensuring that the right personnel have access to the system.
  • Invoice Monitoring: Access a detailed view of all invoices, with options to delete as necessary.
  • Photo Capture: During the invoice creation process, photos can be taken and are displayed on the Super Admin Dashboard for verification and record-keeping.

Admin Dashboard

  • Invoice Creation & Management: Admins can efficiently create and manage invoices, with tools to track the status and details of each invoice.
  • Activity Overview: Provides an at-a-glance view of admin activities, including the number of invoices created and other relevant metrics.
  • Navigation Sidebar: A dedicated sidebar for easy navigation, enabling quick access to essential features and functions.

Invoice Management

The system offers a streamlined process for managing invoices, from creation to deletion. Super Admins can view all invoices, while Admins have the tools to manage the invoices they create. The option to capture and display photos during invoice creation adds an extra layer of security and transparency.
